Zootopia Crime Files Offline

As of December 18th 2017, Disney has removed Zootopia Crime Files from various app stores. The developer behind the game, Hibernum, filed for bankruptcy and have shut down, so they can no longer support the game. We have archived the Android version of the Crime Files game so people can play the game after the content servers for the game go offline. This version can be played on almost any Android device, and Android emulators like BlueStacks to play on a PC.

Installation

  • Download the both files in the folder using the link above.
    1. If it's not enabled, enable installation from unknown sources/sideloading on your Android device.
  • Install the APK.
  • Copy the "Android" folder to the root of the device's storage. NOTE: Android 11 and higher will prevent you from properly adding the files from your phone, for security reasons. You will need a PC to copy the files over USB. (Video)
  • Start the game and enjoy!

Save Game

As of January 15 2018, the in app purchases for the game no longer function, as Hibernum has officially been shut down. To make this game more enjoyable and lessen the grind, we have released a save file that is right after the tutorial case with what should be more than enough currency to progress through the game with boosts.

Save Game Installation

  • Download the save here.
  • Start the game once.
  • Close the game/force stop it.
  • Extract Save.zip to the root of your device/emulator.
  • Navigate to Android/data/com.disney.zootopiacasefiles_goo/files/Save on your Android device/emulator.
  • You should see four files. Two PlayerInventory files, and two PlayerProfile files.
  • Copy the seemingly random sting of numbers and letters after "_" from one of the files.
  • Delete the Player files with those seemingly random strings.
  • Replace "YOURUUID" with the stuff you copied from your original save filename.
  • Start the game, and you should have a save with premium currency that is on the 2nd case, just outside of the tutorial.
